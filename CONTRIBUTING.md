# 贡献指南

感谢你对 ImageCompressor 的关注！

## 如何贡献

### 报告问题
- 使用 GitHub Issues 提交 bug 报告
- 请描述问题现象、复现步骤、期望行为

### 提交代码
1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/xxx`)
3. 提交变更 (`git commit -m "feat: xxx"`)
4. 推送到分支 (`git push origin feature/xxx`)
5. 创建 Pull Request

### 代码风格
- Python: 遵循 PEP 8
- 前端: 使用项目已有的代码风格
- 提交信息: 使用约定式提交格式
